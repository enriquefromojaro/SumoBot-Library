# SumoBot-Library
Librería pensada para el taller de Sumobot de 2015 de AETEL en el Campus Sur.
Permite programar el robot a través de objetos, evitando tener que pensar cómocodificar cosas básicas
  Ejemplo:
    puedes poner esto:
      dist.distance();
    En lugar de esto:
      16746.73 * float(pow(analogRead(pin),-1.2134));
